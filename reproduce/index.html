What the command verifies

The release fails when sources, builds, registries, or artifacts drift.

  • Exact Git state
    Clones four repositories at manifest-pinned commit SHAs.
  • Source identity
    Checks selected canonical files using Git blob hashes.
  • SymPy audit
    Runs tests, full coverage, strict-theory checks, and Lean mapping checks.
  • Registry compiler
    Rebuilds the effective registry and compares its semantic hash.
  • Lean formal layer
    Audits maintained sources, retrieves the pinned Mathlib cache, and runs lake build.
  • Simulation and figure
    Regenerates the deterministic finite-band field, spectrum table, plot, and summary.
  • Published surfaces
    Checks equations, SymPy, Lean, corpus, and traceability synchronization.
  • Final report
    Writes environment versions, verified commits, hashes, counts, and boundaries.
