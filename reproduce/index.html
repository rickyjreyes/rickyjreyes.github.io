What the command verifies

The release fails when commits, sources, builds, counts, or artifacts drift.

  • Exact Git state
    Clones four repositories at manifest-pinned commit SHAs.
  • Source identity
    Checks selected canonical files using Git blob hashes.
  • SymPy audit
    Runs tests, full coverage, strict-theory checks, registry compilation, and reproducibility verification.
  • Lean formal layer
    Audits sources, retrieves the pinned Mathlib cache, and runs lake build.
  • Formal coverage partition
    Requires exactly 62 canonical map entries and 80 registry-only IDs.
  • Published synchronization
    Requires the public Lean surface to display 142, 62, and 80 consistently.
  • Simulation artifacts
    Regenerates the deterministic field, spectrum, figure, and summary and checks SHA-256 values.
  • Final report
    Records commits, source blobs, environment versions, counts, boundaries, and output hashes.
