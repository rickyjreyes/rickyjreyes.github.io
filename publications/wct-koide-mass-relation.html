Richard J. Reyes

Abstract

This work develops a curvature-harmonic model for Koide-like fermion mass triplets. It relates loop geometry, harmonic sidebands, effective spin structure, and scale-free ratios to the charged-lepton value Q = 2/3, while treating the result as a WCT phenomenological derivation.

Plain-language overview

Research question

Can a curvature-harmonic model account for Koide-like fermion mass triplets and the charged-lepton value Q = 2/3?

Main contribution

  • Develops a curvature-harmonic model for Koide-like fermion mass triplets.
  • Relates loop geometry, harmonic sidebands, and scale-free ratios to the charged-lepton value Q = 2/3.

Evidence type

Analytical derivation

Current limitations

The result is treated as a WCT phenomenological derivation, not an independently confirmed prediction of the Standard Model mass spectrum.
