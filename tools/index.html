Visual tools for entering Wave Confinement Theory.

These tools translate the paper archive into hover definitions, equation cards, concept relationships, chronology, dependency structure, and a toy wave-confinement simulator. They are orientation aids, not substitutes for the archived manuscripts.

    5 / Dependency graph

    Which papers depend on which ideas.

    This graph separates “depends on” from “related to,” making the corpus easier for reviewers and AI indexers to classify.

    Click a paper node.

    Legend

    Core papers are entry points. Bridge papers define operators or stability. Application papers extend the framework into data, computation, or audits.

    This is intentionally cautious: a dependency edge says one paper uses another’s definitions or framing.
