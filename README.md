# Richard J. Reyes — Research Site

Source for [rickyjreyes.github.io](https://rickyjreyes.github.io/), a public entry point to Richard J. Reyes's research program.

## Purpose

The site directs new readers to:

- the central Wave Confinement Theory idea;
- the shortest recommended reading path;
- the main research repositories;
- the complete categorized Zenodo publication archive;
- Richard's engineering background and public research profiles.

## Structure

- `index.html` — semantic site content and publication archive
- `styles.css` — responsive visual design
- `script.js` — mobile navigation and publication filtering
- `404.html` — fallback page
- `robots.txt` and `sitemap.xml` — search indexing support

The site is dependency-free and deploys directly through GitHub Pages from the `main` branch.
